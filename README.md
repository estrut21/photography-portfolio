# photography-portfolio
My project is a personal photography portfolio website called "Liz Photography." It's a website where I show my work. The goal is to make it easy for people to view my photos and to contact me for business. 
 
 I wanted a photography website to feel professional. I also wanted to organize my work into categories and start building a real brand around my photography. 
 
 I built this site using HTML and CSS in vs code. I created multiple pages. (Home Gallery About and a Contact page and then I organized the photos into categories like engagement shoots, portraits and events. I used css to style everything control layout and made it feel cohesive; I also worked with asset folders to store images and link them properly across the pages. 
 
 I struggle a lot with layout, like centering and spacing things correctly as well as display. I also had problems with linking pages and organizing files especially when things broke after moving folders. 
 
 From this I learned how file structure actually matters
 how to control layout better with css and how to troubleshoot when things dont work instead of starting over. 
 
 Overall I learned that building a website is a lot of trial and error but each mistake helps me understand how everything is connected
